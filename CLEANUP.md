# Решение проблем с белым экраном

## Быстрое решение

1. **Остановите приложение:** `Ctrl+C` в терминале
2. **Запустите:** `RESTART.bat` или `npm start`
3. **Откройте в режиме инкогнито:** `Ctrl+Shift+N` → `http://localhost:3000`

## Если проблема остается

### 1. Очистка кэша браузера
- **Chrome/Edge:** `Ctrl+Shift+Delete` → Очистить все
- **Firefox:** `Ctrl+Shift+Delete` → Очистить все
- Или используйте режим инкогнито: `Ctrl+Shift+N`

### 2. Проверка консоли браузера
1. Откройте `http://localhost:3000`
2. Нажмите `F12` (Инструменты разработчика)
3. Перейдите на вкладку **Console**
4. Обновите страницу `F5`
5. Посмотрите на ошибки (красный текст)

### 3. Полная очистка проекта
```bash
# Остановить приложение
Ctrl+C

# Удалить node_modules и package-lock.json
rmdir /s node_modules
del package-lock.json

# Переустановить зависимости
npm install

# Запустить заново
npm start
```

### 4. Проверка портов
```bash
# Проверить что запущено на порту 3000
netstat -ano | findstr :3000

# Убить процесс если нужно
taskkill /f /pid [PID_NUMBER]
```

### 5. Альтернативный порт
```bash
# Запуск на другом порту
set PORT=3001 && npm start
```

## Типичные причины белого экрана

1. **Кэш браузера** - самая частая причина
2. **JavaScript ошибки** - проверьте консоль
3. **Неправильные пути к файлам**
4. **Конфликт портов**
5. **Проблемы с зависимостями**

## Проверка работоспособности

Если приложение работает правильно, вы должны увидеть:
- Страницу приветствия с кнопками "Вход" и "Регистрация"
- Возможность зарегистрироваться и войти
- Dashboard с виджетами после входа

## Контакты для помощи

Если ничего не помогает:
1. Сделайте скриншот консоли браузера (F12)
2. Опишите что именно происходит
3. Укажите браузер и версию
